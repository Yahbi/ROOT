# Goal Tracking (OKR Method)

## When to Use
When Yohan sets goals and ROOT needs to track progress.

## OKR Framework
- Objective: Qualitative, inspiring goal
- Key Results: 3-5 measurable outcomes

## Procedure
1. Set Objective (qualitative)
2. Define Key Results (quantitative)
3. Break into weekly tasks
4. Track weekly (score 0.0-1.0)
5. Review monthly
6. Grade quarterly (0.7+ = success)
