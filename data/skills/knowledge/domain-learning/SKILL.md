# Domain Learning

## Purpose
Systematic approach to learning new domains: key concepts, mental models, practice, and teach-back verification.

## Steps
1. Map the domain by identifying the 10-20 core concepts, their relationships, and the vocabulary that experts use
2. Find the best learning resources: one authoritative textbook or course, two practitioner blogs, and one community forum
3. Build mental models by connecting new concepts to things already understood, creating analogies and diagrams
4. Identify the 3-5 most important principles that explain 80% of outcomes in this domain (Pareto learning)
5. Practice through a small hands-on project that forces application of the core concepts in a real context
6. Teach-back by explaining the domain to someone else or writing a summary, identifying gaps in understanding
7. Create a personal reference sheet with key terms, principles, common pitfalls, and links to go deeper

## Output Format
A learning plan with concept map, curated resource list, key principles summary, practice project definition, and a reference sheet for ongoing use.
