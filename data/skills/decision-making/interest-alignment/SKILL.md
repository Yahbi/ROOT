# Interest Alignment

## Purpose
Check if a decision serves Yohan's interests across financial, time, skill, and values dimensions.

## Steps
1. Score the financial impact from -5 to +5 based on near-term cash flow and long-term wealth building
2. Evaluate the time cost relative to current commitments and whether it displaces higher-value work
3. Assess skill development value: does this build capabilities that compound over time or is it a dead end
4. Check values alignment by asking whether this is work Yohan would be proud to talk about
5. Identify who else benefits and whether the incentive structures create genuine alignment or hidden conflicts
6. Stress-test by asking what the worst realistic outcome looks like and whether it is tolerable

## Output Format
A scorecard across the four dimensions (financial, time, skill, values) with an overall alignment verdict and any flags worth noting.
